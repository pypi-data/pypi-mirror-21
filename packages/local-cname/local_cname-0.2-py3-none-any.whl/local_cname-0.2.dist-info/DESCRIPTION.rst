Installation:

.. code-block::

    $ sudo pip install -U local-cname

Usage:

.. code-block::

    $ sudo local-cname FROM_DNS_NAME TO_DNS_NAME

Example:

.. code-block::

    $ sudo local-cname google.com duckduckgo.com
    $ ping google.com
