fast start develop box template

Pre Requirements
----------------

- Vagrant_ && VirtualBox_
- Fabric_

Setup
-----

.. code-block:: shell

    $ pip install falstart


Usage
-----

.. code-block:: shell

    $ cd /your/proj/parent/dir
    $ falstart awesomeprojname
